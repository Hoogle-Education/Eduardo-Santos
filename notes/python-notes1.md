# Recap Python

## Variaveis

- Tipagem fraca

```py
a = 2 # int
a = 2.3 # float
a = 'matheus' # str
a = [1, 2, 3] # list of ...

# checar o tipo
print(type(a))
```

## Print interpolado

```py
nome = 'pedro'
idade = 25
print(f'meu nome eh {nome} e eu tenho {idade} anos')

a = 2
b = 3
print(f'soma = {a + b}')
```

## Entradas

```python
a = input()
b = input()
print(f'soma = {a + b}')
```

## Configurando casas decimais

```py
soma = 23.14159265
print(f"Resultado {soma:.2f}")
```
